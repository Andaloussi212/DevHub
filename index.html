Commandes utiles

Mémo de commandes Git/Linux utiles au quotidien

Commandes affichées : Toutes

Voir l'état d'un projet

Catégorie : Git

Commande : git status

Explication : voir les fichiers modifiés, ajoutés ou non suivis

Quand l'utiliser : Avant d'ajouter des fichiers ou avant de faire un commit

Ajouter tous les fichiers modifiés

Catégorie : Git

Commande : git add .

Explication : Prépare tous les fichiers modifiés pour le prochain commit

Quand l'utiliser : Quand je veux enregistrer l'état actuel du projet

Créer un commit

Catégorie : Git

Commande : git commit -m "message"

Explication : Créer un point de sauvegarde dans l'historique du projet

Quand l'utiliser : Après avoir terminé une étape du projet

Envoyer les changements à GitHub

Catégorie : Git

Commande : git push

Explication : Envoyer mes commits vers le dépôt distant GitHub

Quand l'utiliser : Après un ou plusieurs commits, quand je veux sauvegarder en ligne

Récupérer les changements depuis GitHub

Catégorie : Git

Commande : git pull

Explication : Récupérer les modifications présentes sur GitHub vers mon projet local

Quand l'utiliser : Quand je travaille sur plusieurs machines ou avec plusieurs personnes

Voir l'historique des commits

Catégorie : Git

Commande : git log

Explication : Voir les anciens commits du projet

Quand l'utiliser : Quand je veux comprendre l'évolution du projet

Afficher le dossier actuel

Catégorie : Linux

Commande : pwd

Explication : Affiche le chemin du dossier dans lequel je me trouve

Quand l'utiliser : Quand je veux savoir ou je suis dans le terminal

Lister les fichiers

Catégorie : Linux

Commande : ls

Explication : Affiche les fichiers et dossiers présents dans le dossier actuel

Quand l'utiliser : Quand je veux savoir le contenu d'un dossier depuis le terminal

Changer de dossier

Catégorie : Linux

Commande : cd

Explication : Permet de se déplacer d'un dossier à un autre dans le terminal

Quand l'utiliser : Quand je veux accéder à un autre dossier depuis la ligne de commande

Voir les conteneurs actifs

Catégorie : Docker

Commande : docker ps

Explication : Affiche les conteneurs Docker actuellement en cours d'exécution

Quand l'utiliser : Quand je veux vérifier quels conteneurs sont lancés

Voir les images docker

Catégorie : Docker

Commande : docker images

Explication : Affiche les images Docker présentes sur la machines

Quand l'utiliser : Quand je veux savoir quelles images sont disponibles localement

Voir les logs d'un conteneur

Catégorie : Docker

Commande : docker logs

Explication : Affiche les logs produits par un conteneur Docker

Quand l'utiliser : Quand je veux comprendre ce qui se passe dans un conteneur ou diagnostiquer un problème
